\abstractEN {
    This thesis describes the working mechanisms of the X Window System Protocol
    and two standard protocols defined on top of it, Inter-Client Communication Conventions Manual (ICCCM),
    and Extended Window Manager Hints (EWMH).
    Then, it deals with the design and implementation of a stacking window manager for the X Window System,
    that will be compliant with ICCCM and EWMH.
    The window manager is designed to be small, extensible, and scriptable.
    It is implemented in Go, a modern system programming language, and tested.
}

\abstractCZ {
    Tato práce popisuje princip fungování protokolu X Window System a jeho dvou standardních rozšíření,
    Inter-Client Communication Conventions Manual (ICCCM) a Extended Window Manager Hints (EWMH).
    Práce se následně zabývá návrhem a implementací správce oken pro X Window System,
    který by byl v souladu s ICCCM a EWMH.
    Správce oken je navrhnut tak aby byl minimalistický, snadno rozšiřitelný a dal se ovládat z příkazové řádky a tedy i pomocí skriptů.
    Je implementovaný v Go, moderním programovacím jazyce pro systémové programování, a otestovaný.
}
